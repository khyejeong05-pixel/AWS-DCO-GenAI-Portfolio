# 🚀 AWS DCO 인턴십 직무 이해 & 생성형 AI 활용 학습 포트폴리오

본 저장소는 AWS DCO(Data Center Operations) 직무를 심층적으로 이해하고, 생성형 AI를 활용하여 데이터 센터 운영 및 자동화 프로세스를 실습한 결과물을 정리하는 **학습형 포트폴리오**입니다.

---

## 📌 1. 과정명
* **과정명:** AWS DCO 인턴십 직무 이해를 위한 생성형 AI 활용 및 포트폴리오 제작 과정

## 🎯 2. 포트폴리오 목적
* **직무 이해도 제고:** AWS 데이터 센터 인프라 운영(SOP 분석, 장애 대응 등) 프로세스 이해
* **AI 및 자동화 역량 강화:** 생성형 AI를 활용한 CLI 자동화, 로그 분석, 스크립트 작성 및 대시보드 시각화 구현
* **직무 역량 증명:** 교육 과정 중 도출된 산출물을 체계적으로 구조화하여 DCO 엔지니어로서의 기술적 잠재력 증명

## ⚠️ 3. 교육용 샘플 데이터 사용 안내
> **중요 공지 (Disclaimer)**
> 본 저장소에 포함된 모든 데이터, 로그, SOP(표준운영절차) 및 시나리오는 **실제 AWS의 내부 정보나 기밀 문서가 아닙니다.** > 오직 직무 교육 및 실습 목적으로 구성된 **가상의 교육용 샘플 데이터와 수업 산출물**임을 밝힙니다.

## 📂 4. 폴더별 역할 및 구성 안내

| 폴더명 | 목적 및 주요 포함 내용 |
| :--- | :--- |
| `01_dco_glossary/` | 데이터 센터 운영에 사용되는 주요 기술 용어 및 약어 정리집 |
| `02_dco_profile/` | DCO 직무 분석 및 개인 맞춤형 직무 프로필/역량 기술서 |
| `03_sop_analysis/` | 표준운영절차(SOP) 구조 분석 및 AI를 활용한 요약/해석 실습 |
| `04_environment_setup/` | 자동화 및 스크립트 실행을 위한 개발 환경 구축 가이드 |
| `05_antigravity_cli/` | 가상 환경 또는 특정 CLI 도구(`antigravity`) 활용법 및 명령어 실습 |
| `06_cli_file_automation/` | CLI 기반의 파일 시스템 제어 및 인프라 관리 자동화 스크립트 |
| `07_log_analysis_script/` | 데이터 센터 서버 및 네트워크 로그 파싱/분석 자동화 스크립트 |
| `08_incident_analysis/` | 가상 장애 시나리오 기반 원인 분석(RCA) 및 AI 활용 분석 프로세스 |
| `09_incident_report/` | 장애 대응 완료 후 작성된 가상의 인시던트 리포트 및 회고록 |
| `10_stitch_dco_dashboard/` | 운영 지표 시각화를 위한 대시보드(Stitch 등) 구성 및 템플릿 |
| `11_exam_prep/` | DCO 직무 이해도 평가 및 관련 기술 자격증 대비 학습 노트 |

## 🔒 5. 보안 준수 사항 (Security & Confidentiality)
본 포트폴리오는 보안 가이드라인을 철저히 준수합니다. 다음과 같은 정보는 절대 포함하지 않습니다.
* 실제 기업의 인프라 IP 주소, 자격 증명(Credentials), API Key 등 보안 민감 정보
* 실제 AWS 데이터 센터의 내부 설계도, 실제 SOP 및 기밀 유지 계약(NDA)에 위배되는 모든 정보
* *팁: 실습 중 사용된 모든 환경 변수는 `.env` 파일로 분리하고 `.gitignore` 처리하였습니다.*
